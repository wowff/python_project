# Генератор QR-кода
Приложение для генерации QR-кода с графическим интерфейсом.

Позволяет вводить текст, настраивать размерность модуля в пикселях, уровень коррекции ошибок, изменять цвет генерируемого QR-кода и сохранять готовый результат в формате PNG на диск.

При разработке проекта использовались Python, CSS, PyQt для создания пользовательского интерфейса, фреймворк Segno для генерации QR-кода по введенной строке, а также VisualStudioCode в качестве рабочей IDE.

## Установка (Windows)

1. Клонируйте репозиторий 
  
   ```git clone https://github.com/wowff/python_project.git```

2. Создание виртуального окружения

   ```python -m venv venv```

3. Установка зависимостей   
  
   ```pip install PyQt6```

   ```pip install segno```

4. Запуск приложения

   ```python main.py```

<!--зависимости-->
## Зависимости
Эта программа зависит от интепретатора Python версии 3.14 или выше, PIP 26.1.2 или выше. Необходима версия фреймворков PyQt6 и Segno 1.6.6.
